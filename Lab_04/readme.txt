1. Open the project in Visual Studio. Make sure the relevant references of Navisworks are correctly loaded, otherwise, find them at <Navisworks Installation Path>.
2. Build the project. The binary will be deployed to <Navisworks Installation Path>\Plugins
3. Launch Navisworks. In [View] tab >> [Workspace] panel >> [Windows] drop-down, a dock pane named 'Model DockPane' is available. tick it, the panel appears. 
4. button [Generate Selection Tree] can display the model tree in the tree view. You can make postfix or prefix for the display name in the custom tree view. 
5. select any objects, play with other buttons that override the attributes of the selected objects. button [ResetAll] can restore the attributes to the status of last file saving.

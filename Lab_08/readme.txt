1. Open the project in Visual Studio. Make sure the relevant references of Navisworks are correctly loaded, otherwise, find them at <Navisworks Installation Path>.
2. Build the project. The binary will be deployed to <Navisworks Installation Path>\Plugins
3. Launch Navisworks. 
   3.1 ) In [Tool add-ins *] tab, a button 'Add Custom Properties' is available. Select some objects in advanced, click the button, it will add a new custom property category and property to the objects
   3.2 ) In [Tool add-ins *] tab, a button 'Dump Geometry Primitives' is available. Select some objects in advanced, click the button, it will dump the geometries to 'c:\\temp\\NavisworksObjGeomtries.txt'

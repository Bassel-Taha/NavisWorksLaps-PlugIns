1. Open the project in Visual Studio. Make sure the relevant references of Navisworks are correctly loaded, otherwise, find them at <Navisworks Installation Path>.
2. Build the project. The binary will be deployed to <Navisworks Installation Path>\Plugins
3. Launch Navisworks. In [Tool add-ins *] tab, a button 'Manipulate Viewpoint' is available. 
4. Click the button, a dialog pops out. It can display the saved viewpoints in the tree view, allows to add new saved view or group 2 existing items.
5. It can manipulate the camera by moving or rotation.
